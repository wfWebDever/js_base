# 使用AI快速开发一个包括nextjs apollo graphql tailwindcss的脚手架
最近几个月连续接了几个项目，都是基于nextjs apollo graphql tailwindcss的，
感觉每次都要从头开始搭建一个脚手架，比较麻烦。
于是我想，能不能用AI来帮我快速生成一个脚手架呢？

于是我就试着用AI来生成一个脚手架，结果效果还不错。

时间呢 大约用了半天，修改了大约2次，这个脚手架里面还包括了询问是否包括使用apollo graphql的选项。

所以有了AI编程，以后开发的模式就是你来说需求和想法，AI来帮你生成代码，中间需要你来纠正，随着时间的发展，
这个中间步骤的修正环节也会不断被AI完善。
