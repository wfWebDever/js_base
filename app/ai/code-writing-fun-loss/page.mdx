# 用AI写代码后，乐趣减少了吗

## 1. 排忧解难的乐趣消失了
自从用上了AI工具写代码，遇到问题时可以随时向AI提问，解决方案唾手可得。过去那种查资料、思考、反复尝试的过程变得稀缺，解决难题的成就感和乐趣也随之减少。

## 2. 从0到1的开发过程被简化
以前写一个新功能或项目，需要自己一步步搭建、设计、编码。现在很多时候直接让AI生成代码，开发过程变得像“拼积木”，缺少了从无到有的探索和创造。

## 3. 角色转变为检查者
AI写完代码后，自己的工作变成了检查和修正，像流水线上的工人一样“打螺丝”。虽然效率提升了，但创造力和主动性有所削弱，更多是在做代码审核和微调。
