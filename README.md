# My Blog
my blog website

## 项目描述
这是一个基于 Next.js 的个人博客网站，使用 Material-UI 和 Tailwind CSS 构建现代化的用户界面，支持 Markdown 文章发布，提供良好的阅读体验。

## Link
https://weifengdev.top

## 技术栈
- Next.js (App Router)
- Material-UI (MUI)
- Tailwind CSS
- TypeScript
- ESLint & Prettier

## 环境要求
- Node.js >= 20
- Yarn >= 1.22.19

## 安装步骤
1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
yarn 
```

3. 启动开发服务器
```bash
yarn dev
```

4. 构建生产版本
```bash
yarn build
```

## 项目结构
```
├── app/          #博客文章
├── components/   # 组件文件
```
